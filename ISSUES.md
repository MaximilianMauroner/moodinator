# Issues

Moodinator issue and roadmap state lives in the task tracker, not in this file.

Use this file only as a pointer for agents that discover it during repo scans.
For non-trivial work:

- Create or use a parent task for the idea/report.
- Put report context, links, decisions, assumptions, open questions, and final
  outcome on the parent task.
- Create implementation work as top-level vertical-slice tasks linked to
  the parent.
- Follow the workflow, Definition of Ready, handoff, and merge rules in
  [AGENTS.md](AGENTS.md).
